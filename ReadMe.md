# AIEUnityTutorials

## How To Play

### Objective

Destroy things in the scene by looking at them.

### Controls

#### Desktop

WASD        - Move

Left-Click  - Look, Destruction Ray

#### Mobile

Touch       - Look, Destruction Ray
